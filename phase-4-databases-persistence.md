# Phase 4 — Databases & Persistence

**Goal:** Interact with relational databases correctly — raw JDBC, JPA/Hibernate
ORM, Spring Data repositories, schema migrations, and connection-pool management.
By the end of this phase you can design a persistence layer from scratch and
explain exactly what SQL every layer generates.
